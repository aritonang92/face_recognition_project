## Description
I this project I modified one of [codebasics's project](https://github.com/codebasics/py/tree/master/DataScience/CelebrityFaceRecognition) to build simple Face Classification app for five Selected Indonesia's Actresses, Those are: 
1. Raisa Andriana
2. Isyana Sarasvati
3. Ariel Tatum
4. Maudy Ayunda
5. Pevita Pearce
